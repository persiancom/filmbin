📌 چطور لینک فیلم یا سریال جدید اضافه کنم؟

فایل index.html رو باز کن. برو به پایین صفحه، قسمتی که نوشته // === دیتابیس فیلم‌ها (اینجا لینک جدید اضافه کن) ===

بعد از const moviesData = [ یک کپی از این قالب بردار و اطلاعات جدید رو جایگزین کن:

{
  id: 4,
  title: "اسم فیلم/سریال",
  type: "فیلم", // یا "سریال"
  downloadLink: "https://لینک-دانلود-فیلم.mp4",
  watchLink: "https://لینک-تماشای-آنلاین"
},

💡 نکته: id رو به ترتیب بذار (بعد از آخرین عدد یکی زیاد کن). اگر لینک تماشای آنلاین نداری، بذار watchLink: "" بماند. دکمه‌ها خودکار مخفی می‌شن.

✅ بعد از ذخیره فایل، دوباره آپلودش کن روی GitHub. سایت به‌طور خودکار آپدیت میشه.
